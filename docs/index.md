# 数据集市使用手册


## 什么是风险数据集市?

风险数据集市是一个设计用于管理和分析风险相关数据的系统，包括彭博社数据和其他金融风险指标。

## 文档结构

文档包含以下内容:

- **[数据库文档](database.md)**: 关于数据库架构、表和关系的详细信息
- **[用户指南](handbook)**: 使用风险数据集市的说明
- **[任务计划](schedule)**: 为在风险数据集市工作的开发人员提供的指南

## 开始使用

开始使用风险数据集市:

1. 查看[数据库结构](database.md)以了解可用数据
2. 学习如何使用我们的示例查询来查询数据库
3. 探索任务计划文档以了解数据处理流程
